Monash Course Planner

Plan out your degree with Monash Course Planner. Monash University Course Planner is designed to help you to choose your units for the coming semesters. And takes account into prerequisites as well as prohibitions

Current Features

  • Modern site built with electron
  • Pure.css

Planned Features

  • Takes account of prerequisites and prohibitions
  • Plan for the future
  • Add deferring semester blocks
  • sql database updates from website crawling using python3

Dependancies List

  • npm (node.js)
  • electron

  • Pure.css

  • (sql.js)

  • (python3)

Database is currently stored as a sqlite db, this may be pushed onto a SQL Server in the near future

Usage

Installation

This uses Electron, to install npm package type

sudo npm install

Developement and Testing

To test simply run

npm start

This will run an electron instance
